# Anvil Registry Site

Standalone Next.js promotional and documentation site for Anvil Registry and Anvil Node Base.

## Local development

```bash
pnpm install
pnpm dev
```

Set the repository link with:

```bash
NEXT_PUBLIC_GIT_REPO_URL=https://github.com/your-org/your-repo
```

## Vercel

Deploy with the project root set to this folder:

```text
anvil-site
```

Add `NEXT_PUBLIC_GIT_REPO_URL` in Vercel project settings when the public repo is ready.
